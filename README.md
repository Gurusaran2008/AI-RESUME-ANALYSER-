AI Resume Analyzer is a web-based application that helps users evaluate and improve their resumes using Artificial Intelligence. The system analyzes resume content, identifies strengths and weaknesses, and provides suggestions to enhance resume quality. It helps job seekers create more professional and effective resumes that match industry standards.

Features
       Upload resumes in PDF or text format
       AI-powered resume analysis
       Skill and keyword detection
       Resume score calculation
       Improvement suggestions and feedback
       User-friendly interface

Technologies Used
       Python
       Flask
       HTML, CSS, JavaScript
       Machine Learning / AI
       Git & GitHub
Objective
The main objective of this project is to assist students and job seekers in creating stronger resumes by providing intelligent feedback and recommendations.
